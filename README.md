# Pillow Assignment

Rate Limiting Pillow

## Installation

- Install dependencies

```bash
yarn install
```

- Copy Env file

```shell
cp .env.example .env
```

- Start Project

```bash
yarn start
```

- Start Bot in new terminal window

```bash
ts-node src/bots/read.bot.ts
ts-node src/bots/write.bot.ts
```
